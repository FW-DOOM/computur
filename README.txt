PRANK PACK - Computur
=====================

PRANKS INCLUDED:
----------------

1. broken_screen.py
   - Makes the screen look shattered/cracked with impact points
   - Press ESC to close

2. error_storm.py
   - Floods the screen with hilarious fake Windows error popups
   - They keep spawning randomly
   - Close the first box that appears to stop it

3. fake_update.py
   - Fullscreen fake Windows Update screen stuck at 32%
   - Funny fake status messages like "Downloading more RAM..."
   - Press ESC three times quickly to escape

4. mouse_chaos.py
   - Randomly teleports the mouse cursor all over the screen
   - A small stop button stays on screen so YOU can stop it
   - Requires: pip install pyautogui

HOW TO COMPILE TO .EXE:
------------------------
1. Make sure Python is installed: https://python.org
2. Double-click compile_all.bat
3. Find the .exe files in the dist\ folder

REQUIREMENTS:
-------------
- Python 3.x
- pip install pyinstaller pyautogui
  (compile_all.bat does this automatically)

NOTE: These are harmless visual pranks. Nothing is deleted,
installed, or changed on the computer permanently.
