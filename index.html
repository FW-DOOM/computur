⚙️  Extension Install Steps

1. Download or clone this repo  →  📦 Download ZIP
2. Unzip it, then open Chrome and go to  chrome://extensions
3. Turn on Developer mode (toggle top-right)
4. Click Load unpacked → select the chrome_version/extension folder
5. Click the extension icon in Chrome toolbar → Launch Security Alert
